# Визуальный референс MVP

Файл `analytics-dashboard-reference.svg` — сохранённый пример визуализации итогов аналитики.

Что можно перенести в будущий MVP:

- тёмная профессиональная тема;
- боковая навигация по этапам аналитики;
- карточки с ключевыми выводами;
- цветовые индикаторы статусов и полноты данных;
- компактная матрица результатов;
- визуальное разделение выводов, рисков и следующих действий.

Референс задаёт направление интерфейса, но не определяет объём функций MVP.
