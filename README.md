# Redstar MVP Intensive

Учебный проект интенсива «От идеи до работающего MVP».

## Результаты

- `results/PRD.md` — продуктовые требования внутреннего сервиса аналитики;
- `results/my-landing-page-text.md` — согласованный текст лендинга веб-студии;
- `results/landing.html` — адаптивный HTML-лендинг Редстар;
- `results/about-rat.md` — упражнение по Riskiest Assumption Test;
- `results/about-site-analysis.md` — описание первичной аналитики сайта.

## Просмотр лендинга

Откройте `index.html` или опубликованную GitHub Pages-страницу репозитория.

## Структура

- `prompts/` — инструкции интенсива;
- `references/` — визуальные референсы;
- `results/` — созданные на интенсиве материалы;
- `what-to-do.md` — программа занятий.
